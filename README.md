📖 Overview

This project is a responsive frontend clone of Amazon designed to replicate the look and feel of the world's largest e-commerce platform.

The application demonstrates modern web development concepts including responsive layouts, navigation bars, product sections, banners, cards, and interactive UI components.

---

✨ Features

✅ Amazon-inspired Homepage

✅ Responsive Navigation Bar

✅ Search Bar UI

✅ Product Categories Section

✅ Hero Banner

✅ Shopping Cards Layout

✅ Footer Section

✅ Mobile-Friendly Design
